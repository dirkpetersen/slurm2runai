# Install Using Base Command Manager (BCM)

{% hint style="info" %}
**Note**

NVIDIA Run:ai version 2.25 is not yet available through NVIDIA Base Command Manager (BCM).

NVIDIA Base Command Manager (BCM) currently supports NVIDIA Run:ai versions [2.23](/self-hosted/2.23/getting-started/installation/bcm-install.md) and [2.22](/self-hosted/2.22/getting-started/installation/bcm-install.md). If you are deploying through BCM, refer to the BCM-specific installation instructions for these supported releases.
{% endhint %}
